## How to use

* clone this repo
* run `npm install` to install all dependencies
* run `npm start` to build the project
* to compile SCSS to CSS open new terminal and run `npm run watch-css`
* visit `http://localhost:3000/` in the browser
* follow step by step

Have fun!
